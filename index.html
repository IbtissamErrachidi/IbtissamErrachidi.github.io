Image Compression with the New .irm Extension

A project to develop an advanced image compression algorithm using the new .irm extension.

View the Interface

Features

Choose Image

This feature allows users to browse their system files to select an image for compression. Once the image is selected, its complete file path is displayed in the "Image Path" text area, and the image name is shown in the "Image Name" text area. The selected image is also displayed in the application window.

Compress Button

This button initiates the compression process for the selected image. Upon completion of compression, information regarding the compression rate and Mean Squared Error (MSE) are displayed in their respective text areas.

Decompress Button

This button displays the compressed and decompressed image in the application window. It becomes active once the compression process is completed and the decompressed image is available.

Information Button

This button displays the original image's information such as size in the application.

Show IRM Image File Button

This button allows users to view the decompressed IRM image alongside the original image in the application window. It becomes active once the decompression process is completed and the IRM image is available.

Clear Button

This button allows the user to clear all displayed information in the interface, including the image path, image name, compression rate information, Mean Squared Error (MSE), and displayed images. This enables the user to start the process anew with a fresh image.

About

The inspiration behind the project stemmed from the desire to amalgamate the functionalities of existing image extensions, namely JPG and GIF. This fusion was driven by the necessity to create a more versatile and efficient image compression algorithm capable of retaining both the high-quality compression of JPG and the animation capabilities of GIF, thus offering users a comprehensive solution for image compression needs.
